Setup Instructions

Follow these steps to set up and access the remote server desktop application:

  1. Download and install the remote desktop client software provided by IT.
  2. Open the remote desktop client and enter the server address provided by your IT department.
  3. Enter your credentials to connect to the server.
  4. Once connected, navigate to the file system to access client documents securely.
  5. When finished, log out of the remote desktop session to ensure security.

Best Practices

For secure and efficient use of the remote server desktop application:

  • Always log out after use: Ensure you log out of the remote server to protect client data.
  • Use strong passwords: Follow IT guidelines for creating and updating your passwords.
  • Maintain regular backups: Back up important files to the cloud or an external drive as directed by IT.
